# دليل التوافقية

## معايير التوافقية

### الهاتف (Mobile)
- عرض كامل للنص
- صور محسّنة للشاشات الصغيرة
- Navigation محسّنة للمس

### الأجهزة اللوحية (Tablet)
- تخطيط محسّن
- شبكات 2-3 أعمدة

### سطح المكتب (Desktop)
- تخطيط كامل
- شبكات متعددة الأعمدة
- تفاعلات hover محسّنة

## الاختبار
اختبر على:
- iPhone SE (375px)
- iPhone 12 (390px)
- iPad (768px)
- Desktop (1920px)
