# هيكل المشروع

## المجلدات الرئيسية

### /app
- صفحات التطبيق (Pages) ومسارات الـ API
- الـ layouts والـ root components

### /components
- مكونات React قابلة لإعادة الاستخدام
- مكونات الـ UI والتفاعلات

### /lib
- دوال الـ utilities والـ helpers
- إعدادات خارجية (Supabase, etc)

### /contexts
- React contexts (Language, Theme, etc)

### /public
- ملفات الصور (231+ صورة)
  - `/images/` - صور الخدمات والفريق
  - `/gallery/` - معرض الأثاث والتصاميم
  - `/projects/` - صور المشاريع

### /docs
- التوثيق الشاملة للمشروع
